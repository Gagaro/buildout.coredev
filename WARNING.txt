This is a mr.developer based buildout for Plone 3.3. The official buildout
for releases is at https://svn.plone.org/svn/plone/plonenext/3.3 any updates
to versions need to be done there, otherwise they won't go into a release.

You should call bin/sync-with-plonenext to update this buildout if there are
changes in plonenext/3.3.
